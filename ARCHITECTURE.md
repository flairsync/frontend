# Architecture Rules (Frontend)

1. **Components**: All React components (UI elements, layout parts, pages components) MUST be located under the `components/` directory (e.g., `components/feed/`, `components/explore/`).
2. **Features**: All API calls, services, TanStack hooks, and store configurations MUST be located under the `features/` directory (e.g., `features/discovery/`, `features/business/`).
